# はじめに
本ページは書籍「作りながら丁寧に学ぶPythonプログラミング入門」のサポートページです。
本ページでは、書籍で利用するデータやインストール方法、プログラミング初心者をサポートする内容を紹介します。

サポートページで紹介するURLなどは予告なく変更される場合があります。
